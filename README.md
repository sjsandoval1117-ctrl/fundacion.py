# 🐾 Patitas Felices

## 📌 Descripción del proyecto
Patitas Felices es una página web estática creada con HTML y CSS. Su objetivo es promover la adopción de animales rescatados, mostrando sus perfiles, permitiendo contacto, donaciones y registro de voluntarios.

---

## 💻 Tecnologías utilizadas
- HTML5
- CSS3

---

## 📱 Diseño responsivo
El sitio es completamente responsivo y se adapta a diferentes dispositivos mediante media queries:

- 💻 Escritorio (Desktop)
- 📱 Tablet
- 📱 Móvil

---

## 🎠 Componentes del proyecto
- Tarjetas de animales en adopción
- Perfiles individuales de cada animal
- Formulario de contacto
- Formulario de donaciones
- Sección de voluntariado
- Galería de imágenes
- Navegación entre páginas

---

## 🚀 Instrucciones de uso
1. Descargar o clonar el repositorio
2. Abrir el archivo `index.html` en un navegador
3. Navegar entre las secciones desde el menú
